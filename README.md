<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">高性能物理 3D 模拟器，支持：</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">室内/室外空间的 3D 扫描（内置支持</font></font><a href="https://aihabitat.org/datasets/hm3d/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">HM3D</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><a href="https://niessner.github.io/Matterport/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">MatterPort3D</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><a href="http://gibsonenv.stanford.edu/database/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Gibson</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><a href="https://github.com/facebookresearch/Replica-Dataset"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Replica</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">和其他数据集）</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">空间和分段刚性物体的 CAD 模型（例如</font></font><a href="https://aihabitat.org/datasets/replica_cad/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">ReplicaCAD</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><a href="https://www.ycbbenchmarks.com/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">YCB</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">、</font></font><a href="https://app.ignitionrobotics.org/GoogleResearch/fuel/collections/Google%20Scanned%20Objects" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Google Scanned Objects</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">），</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">可配置传感器（RGB-D 摄像头、自我运动感应）</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">通过 URDF 描述的机器人（像</font></font><a href="http://docs.fetchrobotics.com/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Fetch这样的移动机械手，像</font></font></a><font style="vertical-align: inherit;"></font><a href="https://www.franka.de/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Franka</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这样的固定底座手臂</font><font style="vertical-align: inherit;">，像</font></font><a href="https://www.unitree.com/products/aliengo/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">AlienGo</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这样的四足动物），</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">刚体力学（通过</font></font><a href="https://github.com/bulletphysics/bullet3"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Bullet</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">）。</font></font></li>
</ul>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Habitat 的设计理念是优先考虑仿真速度而不是仿真功能的广度。</font><font style="vertical-align: inherit;">当渲染 Matterport3D 数据集的场景时，Habitat-Sim 在单线程运行时可实现每秒数千帧 (FPS)，并在单个 GPU 上的多进程上达到超过 10,000 FPS。</font><font style="vertical-align: inherit;">Habitat-Sim 模拟 Fetch 机器人以每秒超过 8,000 步 (SPS) 的速度在 ReplicaCAD 场景中交互，其中每个“步”涉及渲染 1 个 RGBD 观察（128×128 像素）和 1/30 秒的刚体动力学。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Habitat-Sim 通常与
</font></font><a href="https://github.com/facebookresearch/habitat-lab"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Habitat-Lab</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">一起使用，Habitat-Lab 是一个模块化高级库，用于嵌入式 AI 中的端到端实验 - 定义嵌入式 AI 任务（例如导航、指令跟踪、问答）、训练代理（通过模仿或强化学习，或者像经典的 SensePlanAct 管道那样根本不学习），并使用标准指标对定义的任务的性能进行基准测试。</font></font></p>
